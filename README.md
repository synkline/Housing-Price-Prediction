# Housing-Price-Prediction

A Simple Desktop GUI Application written in Python which uses Linear Regression and Random Forest models to Predict Housing Prices using the dataset "Housing Prices Prediction for Metropolitan Areas of India" from Kaggle.

## Technologies Used:
- **Language:** Python.
- **Libraries:** Pandas, NumPy, Scikit-learn, Joblib.
- **Interface:** Tkinter.

## Project Structure:
- `app_tkinter.py`: The main GUI application for users.
- `main.py`: The core script for training or running the logic.
- `models/`: Contains the trained models 
- `archive/`: Raw datasets and historical data.

## How To Run:
1. Clone the Repository.
   ```
   git clone https://github.com/synkline/Housing-Price-Prediction.git

2. Navigate to the Folder.
   ```
   cd Housing-Price-Prediction

3. Run the Application.
   ```
   python app_tkinter.py
